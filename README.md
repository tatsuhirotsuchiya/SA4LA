# SA4LA
Simulated annealing for obtaining (\overline{1},t)-locating arrays

# Usage 
`java -jar program.jar inputfile`

# Inputfile format
    [strength]
    [#parameters]
    [#numOfValues1 #numOfValues2 ... ]
## Example
    2
    10
    2 2 2 2 2 2 3 3 3 4
