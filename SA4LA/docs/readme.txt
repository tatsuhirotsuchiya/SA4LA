（古い）

実行コマンド
java -jar ./saforlav19,jar [入力ファイル]

入力ファイル

テキストファイルに
[強度]\n
[パラメータ数]\n
[パラメータ値]
の順番で書く

例:
2
10
2 2 2 2 2 2 3 3 3 4

V19, v20の違い
v19:(1,t)-LAの生成のみ (dを2以上にしても動くが、正しいLAを生成できないため機能を封印)
    DC研究会で使用したプログラムはこちら

v20:(d,t)-LAの生成 (最大d個までの不具合の原因となるインタラクションを特定可能)
    retriesを1から2に変更

retries
探索の結果(見つけた中での最小テストケース数)と、iterationを2倍にし再度探索した結果が同じだった場合のリトライの回数
1以下:   探索を終了
n(2以上)：n-1回、より小さな結果を発見できるまで、iterationを2倍にしつつ探索を繰り返す
        見つからなかった場合、探索を終了

なお、より小さな結果を発見できた場合、リトライ数をリセットする。
