実験1 
目的：
 baseline algorithm　と提案法を比較する
　パラメータのチューニングを行う

ベンチマーク
spin-s-2をつかっておこなう．

実行コマンド
java -jar file.jar [--baseline] -w [重み] -t [温度] -i　[イタレーション] -s [乱数の種] spin-s-2

--baseline
つける場合ー＞ baselineのアルゴリズム
つけない場合　－＞　提案アルゴリズム

重み (カバーされていないインタラクションを，目的関数でどの程度評価するか)
0.5 1.0 2.0 4.0

温度　(初期温度)
0.5 1.0 2.0 4.0

イタレーション
2048 4096 8192 16384

乱数の種
0 1 2 3 4

２つのアルゴリズムそれぞれについて，（重み，温度，イタレーション）を固定して，5回実行
２つのアルゴリズムそれぞれについて，（平均時間，平均rowsの数)が64点えられる．
